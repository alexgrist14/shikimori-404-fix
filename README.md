# Shikimori 404 Fix — возврат удалённых 18+ тайтлов

Юзерскрипт для **Shikimori** ([shikimori.one](https://shikimori.one) / shikimori.io / shiki.one), который **восстанавливает удалённые 18+ тайтлы** (аниме, мангу, ранобэ), скрытые администрацией сайта.

Когда Shikimori убрал весь контент с рейтингом **Rx / Hentai**, страницы таких тайтлов отдают **404** и пропадают из поиска. Скрипт возвращает их прямо в интерфейсе сайта.

> Ключевые слова: shikimori 404 fix, шикимори восстановить удалённые тайтлы, шикимори 18+, hentai, Rx, эротика, возврат хентая, shikimori hentai userscript, tampermonkey.

---

## 🚀 Установка

1. Поставьте менеджер юзерскриптов: **[Tampermonkey](https://www.tampermonkey.net/)** (Chrome/Edge/Firefox) или Violentmonkey.
2. Откройте файл — Tampermonkey предложит установку:

   **➡️ [Установить 404FIX.bundle.user.js](https://raw.githubusercontent.com/alexgrist14/shikimori-404-fix/main/404FIX.bundle.user.js)**

3. Готово. Заходите на удалённый тайтл — страница восстановится.

> Ставьте именно **`404FIX.bundle.user.js`** — он самодостаточный (список удалённых тайтлов вшит внутрь). Файл `404FIX.user.js` — исходник для разработки.

Автообновление настроено на этот же файл (`@updateURL`).

## ✨ Возможности

- **Восстановление 404-страниц** аниме/манги/ранобэ — мгновенно, без перезагрузки (перехват клика по удалённым ссылкам).
- **Постеры** где их вырезали: на странице тайтла, в «Похожем» и «Связанном», в каталоге, списках, **истории профиля** и во всплывающих карточках (снятие цензур-блюра).
- **Комментарии** — корректный счётчик и подгрузка (3 по умолчанию, далее по 10).
- **Поиск** по удалённым тайтлам в глобальном поиске сайта (`censored: false`).
- **Оценка и прогресс** — нативный виджет статуса/звёзд/серий (можно оценивать и отмечать просмотренное).
- **Префетч** данных по наведению/нажатию — открывается почти мгновенно.
- Корректный блок «Информация» у манги, прогресс-бар загрузки, рабочая кнопка «Назад».

Источники данных: GraphQL/REST API самого Shikimori + [Jikan](https://jikan.moe/) (зеркало MyAnimeList) для постеров.

## 🛠️ Для разработчиков

```
404FIX.user.js          — исходник
404FIX.bundle.user.js   — собранный бандл (встроен deleted-ids.json) — этот и устанавливать
deleted-ids.json        — список ID удалённых 18+ тайтлов (MAL == Shikimori ID)
tools/generate-deleted-ids.mjs — генератор списка через Jikan
tools/build.mjs         — сборщик бандла
.verify/                — тесты (Node + Playwright)
```

Обновить список удалённых тайтлов и пересобрать:

```bash
node tools/generate-deleted-ids.mjs   # перекачивает deleted-ids.json (~14 мин, Jikan)
node tools/build.mjs                   # пересобирает 404FIX.bundle.user.js
```

---

## ⚠️ Ограничения

- **Отзывы и рецензии** удалённых тайтлов восстановить нельзя: Shikimori не отдаёт их список через API для удалённых страниц.
- Список 18+ тайтлов основан на рейтинге MAL (Rx / Hentai) и периодически устаревает — пересобирайте `deleted-ids.json`.
- Скрипт ничего не публикует и не отправляет ваши данные третьим лицам; запросы идут только к Shikimori и публичному Jikan API.
